## ALM Simulator

### Overview
This projects implements an $\textbf{\textit{ALM simulator}}$ containing following structures:
* Interest rate simulation via **Hull-White model**
* Monte carlo simulation of assets and liabilities
* Yield curve reconstruction using **PCA**
* Equity, funding ratio, duration gap analysis on both base and stressed curve
* Related visualization for each analysis

### How to Run

    pip install -r requirements.txt
    python main.py

Then open notebooks to explore results
